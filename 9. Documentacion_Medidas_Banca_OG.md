## 1. Medidas de Soporte

Medidas base que son referenciadas internamente por otras medidas del dashboard.

---

### `Ultima fecha t2 y t3`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Obtiene la fecha más reciente disponible en la tabla de estados financieros. Es la medida ancla de tiempo utilizada por casi todos los KPIs para filtrar al último mes cargado.
- **Fórmula DAX:**
```dax
CALCULATE(
    MAX(FCT_estado_financiero_t2[fecha_mes]), 
    ALL(FCT_estado_financiero_t2[fecha_mes]) 
)
```

---

### `Saldo Capital (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Suma simple del saldo de capital de todos los créditos en la tabla de créditos.
- **Fórmula DAX:**
```dax
SUM(FCT_creditos_t1[saldo_capital])
```

---

### `Cartera bruta (SUM) t1`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Suma del saldo de capital desde la tabla de créditos individuales (FCT_creditos_t1). Usada como base para cálculos de costo de riesgo.
- **Fórmula DAX:**
```dax
SUM(FCT_creditos_t1[saldo_capital])
```

---

### `Numero de creditos t1 (Total)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Cuenta el número de créditos distintos en la tabla de créditos.
- **Fórmula DAX:**
```dax
DISTINCTCOUNT(FCT_creditos_t1[id_credito])
```

---

### `Numero de creditos t4 (Total)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Cuenta el número de créditos distintos en la tabla de historial de pagos.
- **Fórmula DAX:**
```dax
DISTINCTCOUNT(FCT_historial_pagos_t4[id_credito])
```

---

### `Tasa Mora (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Suma la tasa de mora desde la tabla de balance general. Medida base para calcular la tasa de mora del último mes y comparativas.
- **Fórmula DAX:**
```dax
SUM(FCT_balance_general_t3[tasa_mora_pct_t1])
```

---

### `Cartera en Mora (GENERAL)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Calcula el saldo de capital de los créditos con calificación de riesgo alto: Pérdida, Deficiente o Dudoso.
- **Fórmula DAX:**
```dax
CALCULATE(
    SUM(FCT_creditos_t1[saldo_capital]),
    DIM_Calificacion[calificacion] in {"Pérdida","Deficiente","Dudoso"}
)
```

---

### `Cartera en Vigente (GENERAL)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Calcula el saldo de capital de créditos con calificación Normal o Vigente.
- **Fórmula DAX:**
```dax
CALCULATE(
    [Saldo Capital (SUM)],
    DIM_Calificacion[calificacion] IN {"Normal", "Vigente"}
)
```

---

### `Zero`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** Medias_soporte
- **Descripción:** Constante fija en 0. Usada como referencia de línea base en visualizaciones.
- **Fórmula DAX:**
```dax
0
```

---

### `Filtros`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Descripción:** Texto estático "Filtro". Usada como etiqueta de sección en el dashboard.
- **Fórmula DAX:**
```dax
"Filtro"
```

---

### `Ultima actualizacion 2`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Descripción:** Retorna la fecha máxima del balance general ignorando filtros del calendario. Muestra la última actualización de datos disponible.
- **Formato:** Fecha corta
- **Fórmula DAX:**
```dax
VAR maxfecha = CALCULATE(
    MAX(FCT_balance_general_t3[fecha_mes]),
    ALL(DIM_Calendario[Fecha])
)
RETURN maxfecha
```

## 2. KPI - ROE (Retorno sobre Patrimonio)

---

### `Utilidad neta (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Suma de la utilidad neta del período desde el estado financiero.
- **Fórmula DAX:**
```dax
SUM(FCT_estado_financiero_t2[utilidad_neta])
```

---

### `Patrimonio total (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Suma el patrimonio total desde el balance general.
- **Fórmula DAX:**
```dax
SUM(FCT_balance_general_t3[total_patrimonio])
```

---

### `ROE % (GENERAL)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Calcula el ROE general como el cociente entre utilidad neta y patrimonio total. Base para el KPI del último mes.
- **Formato:** Porcentaje con 2 decimales
- **Fórmula DAX:**
```dax
DIVIDE(
    [Utilidad neta (SUM)],
    [Patrimonio total (SUM)]
)
```

---

### `ROE`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** ROE filtrado al último mes disponible de datos. Es el valor que se muestra en el KPI principal del dashboard.
- **Formato:** Porcentaje con 2 decimales
- **Fórmula DAX:**
```dax
VAR UltimaFechaCargada = [Ultima fecha t2 y t3]
VAR calculo =
CALCULATE(
    [ROE % (GENERAL)],
    DIM_Calendario[Fecha] = UltimaFechaCargada
)
RETURN calculo
```

---

### `ROE % (Mes anterior)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Calcula el ROE del mes inmediatamente anterior al último mes cargado. Permite comparación periodo a periodo.
- **Fórmula DAX:**
```dax
VAR FechaActual = [Ultima fecha t2 y t3]
VAR FechaAnterior =
    CALCULATE(
        MAX(FCT_estado_financiero_t2[fecha_mes]),
        FCT_estado_financiero_t2[fecha_mes] < FechaActual,
        ALL(FCT_estado_financiero_t2)
    )
RETURN
CALCULATE(
    [ROE % (GENERAL)],
    DIM_Calendario[Fecha] = FechaAnterior
)
```

---

### `ROE DIFF`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Diferencia entre el ROE actual y el del mes anterior. Indica si el indicador mejoró o empeoró.
- **Fórmula DAX:**
```dax
[ROE] - [ROE % (Mes anterior)]
```

---

### `ROE ETIQUETA`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Texto con la variación del ROE formateada, acompañada de emoji de flecha arriba (🡹) o abajo (🡻) según la tendencia.
- **Fórmula DAX:**
```dax
VAR diferencia = FORMAT([ROE DIFF], "0.00%")
RETURN 
IF(
    [ROA DIFF] > 0,
    "+" & diferencia & UNICHAR(129145),
    diferencia & UNICHAR(129147)
)
```

---

### `ROE ETIQUETA - COLOR`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Devuelve el código de color (#2ECC71 verde / #E74C3C rojo) para colorear condicionalmente la etiqueta del ROE según su tendencia.
- **Fórmula DAX:**
```dax
IF([ROA DIFF] > 0, "#2ECC71", "#E74C3C")
```

---

### `ROE Mes anteror - Etiqueta`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** ROE KPI
- **Descripción:** Genera el texto "vs X.XX% Mes Anterior" para mostrar el dato comparativo debajo del KPI.
- **Fórmula DAX:**
```dax
VAR RoaANt = FORMAT([ROE % (Mes anterior)], "0.00%")
RETURN "vs " & RoaANt & " Mes Anterior"
```

## 4. KPI - NIM (Margen de Interés Neto)

---

### `Margen Financiero Bruto (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Suma del margen financiero bruto del estado de resultados.
- **Fórmula DAX:**
```dax
SUM(FCT_estado_financiero_t2[margen_financiero_bruto])
```

---

### `Activos Productivos (SUM)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Suma de activos productivos (cartera bruta + inversiones + operaciones interbancarias activo).
- **Fórmula DAX:**
```dax
SUM(FCT_balance_general_t3[cartera_bruta]) + 
SUM(FCT_balance_general_t3[inversiones]) + 
SUM(FCT_balance_general_t3[operaciones_interbancarias_activo])
```

---

### `Activos productivos (ULTIMO MES)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Activos productivos filtrados al último mes disponible.
- **Fórmula DAX:**
```dax
VAR UltimaFechaCargada = [Ultima fecha t2 y t3]
VAR calculo =
CALCULATE(
    [Activos Productivos (SUM)],
    DIM_Calendario[Fecha] = UltimaFechaCargada
)
RETURN calculo
```

---

### `Activos productivos (MES ANTERIOR)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Activos productivos del mes anterior al último cargado. Usado para el promedio.
- **Fórmula DAX:**
```dax
VAR FechaActual = [Ultima fecha t2 y t3]
VAR FechaAnterior = 
    CALCULATE(
        MAX(FCT_estado_financiero_t2[fecha_mes]),
        FCT_estado_financiero_t2[fecha_mes] < FechaActual,
        REMOVEFILTERS(FCT_estado_financiero_t2)
    )
VAR calculo =
CALCULATE(
    [Activos Productivos (SUM)],
    DIM_Calendario[Fecha] = FechaAnterior
)
RETURN calculo
```

---

### `Activos Productivos Promedio (AVG)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Promedio de activos productivos entre el mes actual y el anterior. Denominador del NIM.
- **Fórmula DAX:**
```dax
CALCULATE(
    ([Activos productivos (ULTIMO MES)] + [Activos productivos (MES ANTERIOR)]) / 2
)
```

---

### `Margen Financiero Bruto (ULTIMO MES)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Margen financiero bruto filtrado al último mes disponible.
- **Fórmula DAX:**
```dax
VAR UltimaFechaCargada = [Ultima fecha t2 y t3]
VAR calculo =

    [Margen Financiero Bruto (SUM)],

)
RETURN calculo
```

---

### `NIM`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Margen de Interés Neto. Mide la eficiencia en la generación de ingresos financieros respecto a los activos productivos promedio.
- **Formato:** Porcentaje con 2 decimales
- **Fórmula DAX:**
```dax
DIVIDE(
    [Margen Financiero Bruto (ULTIMO MES)],
    [Activos Productivos Promedio (AVG)]
)
```

---

### `NIM (Mes anterior)`
- **Tabla:** `1_DS_Resumen Ejecutivo`
- **Carpeta:** NIM KPI
- **Descripción:** Calcula el NIM del mes anterior para comparación. Utiliza los dos meses previos para calcular el promedio de activos productivos del período anterior.
- **Formato:** Porcentaje con 2 decimales
- **Fórmula DAX:**
```dax
VAR FechaSeleccionada = [Ultima fecha t2 y t3]
VAR FechaAnteriorCalculo =
    CALCULATE (
        MAX ( FCT_estado_financiero_t2[fecha_mes] ),
        FCT_estado_financiero_t2[fecha_mes] < FechaSeleccionada,
        REMOVEFILTERS ( FCT_estado_financiero_t2 )
    )
VAR FechaMesAnterior =
    CALCULATE (
        MAX ( FCT_estado_financiero_t2[fecha_mes] ),
        FCT_estado_financiero_t2[fecha_mes] < FechaAnteriorCalculo,
        REMOVEFILTERS ( FCT_estado_financiero_t2 )
    )
VAR Margen =
    CALCULATE (
        [Margen Financiero Bruto (SUM)],
        DIM_Calendario[Fecha] = FechaAnteriorCalculo
    )
VAR AP_Actual =
    CALCULATE (
        [Activos Productivos (SUM)],
        DIM_Calendario[Fecha] = FechaAnteriorCalculo
    )
VAR AP_Anterior =
    CALCULATE (
        [Activos Productivos (SUM)],
        DIM_Calendario[Fecha] = FechaMesAnterior
    )
RETURN
    DIVIDE ( Margen, ( AP_Actual + AP_Anterior ) / 2, 0 )
```
